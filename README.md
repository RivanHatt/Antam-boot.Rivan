#  Antam Boot System
**Antam Boot System** adalah dashboard kendali berbasis web yang dirancang untuk manajemen sistem yang cepat, responsif, dan elegan.

---

##  Fitur Utama
* **Real-time Dashboard:** Monitoring status sistem secara langsung.
* **Responsive Design:** Tampilan optimal di perangkat mobile (Acode) maupun desktop.
* **Lightweight:** Kode bersih dan ringan tanpa beban library yang tidak perlu.

##  Teknologi yang Digunakan
* **HTML5:** Struktur utama aplikasi.
* **CSS3:** Styling dengan tema gelap (dark mode) yang futuristik.
* **JavaScript:** Logika fungsional untuk sistem boot.

##  Cara Penggunaan
1.  Buka link GitHub Pages kamu: `https://usernamekamus.github.io/antam-boot-system/`
2.  Gunakan panel kontrol untuk mengelola sistem.
3.  Edit kode melalui **Acode** atau langsung di repository ini untuk pengembangan lebih lanjut.
